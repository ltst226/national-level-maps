Breweries Across the U.S.

This map was created while taking MAP 671: Introduction to New Mapping at the University of Kentucky. This map was created halfway through the course. This map shows hexbins, that are categorized into 5 levels. These levels show how many breweries are located within the hexbin. For this map, the hexbins were set to 50 km diameters.

This map was created in QGIS by taking in .csv data provided by the professor, which they received from the listed source. This .csv data was converted into point data and then created new polygon shapefiles, the hexbins, that will be used to count data points within each hexbin.

  • See Map: Hexbin Map of Breweries with the United States
  • Data Source: The POI Factory. CSV data file: poifactory-breweries.csv

Cancer Rates in the U.S. by Counties

This map was created in the same class as the map above; however, further along in the class. This map was used in coordination with CartoDB for making online, interactive maps.

Data for this map was retrieved from the State Cancer Profiles website, downloaded onto our local machines, used withing QGIS, and integrated with CartoDB for making an interactive map.

  • See the map: Cancer Rates in the U.S. by Counties
  • Data Source: State Cancer Profiles

Power Plants Across the U.S.

This map was created in MAP 672 while learning javascript programming. The programming language was written inside a program called Brackets. We used this information along with Github to upload and share our data.

This proportional symbol map was created with data gathered from the Energy Information Administration. We used a seperate file to load into our javascript programming to link all the symbols. The program was then created to become user friendly and included writting programming to add popups.
